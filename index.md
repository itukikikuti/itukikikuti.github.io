# GitHub Pagesを試しに使ってみました
まあJekyllなんで個人的には新鮮味は何もないんですが、これだけ簡単にサイトを公開できるのはいいですよね。レンタルサーバーとかなくていいし、レスポンスいいし。

GitHubって便利ですね。しかしGitクライアントは何が一番いいんでしょうか。SSHさえ作ってしまえば後は使いやすいSourceTree？アカウントの連携が簡単だけど動作がもっさりでコミットをチェックアウトできないGitKraken？これ以上ないくらいシンプルだけどこれまたコミットをチェックアウトできないGitHub Desktop？まだ使ったことがないので分からないけどForkというのもある。Forkは動作もサクサクでコミットのチェックアウトもできるけどアカウント周りとかはどんな感じなんだろう。
